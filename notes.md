Command,Iska Matlab (Purpose),Kyun aur Kab Chalani Hai?
touch notes.md,Nayi file banane ke liye.,Jab tumhein koi nayi notebook ya notes wali file banani ho.
python --version,Python engine check karne ke liye.,Ye dekhne ke liye ke AI ka dimaag (Engine) sahi version par hai ya nahi.
pip --version,Delivery boy (Installer) check karne ke liye.,Jab tumhein AI ki nayi libraries ya tools download karne hon.
git add .,Saari files ko pack (Stage) karne ke liye.,Jab tum koi change kar lo aur usay save karne ke liye tayyar karna ho.
"git commit -m ""message""",Pack par label (Message) lagane ke liye.,Taake baad mein yaad rahe ke is save mein tumne kya kaam kiya tha.
git push,Cloud (GitHub) par hamesha ke liye save karne ke liye.,Apne kaam ko internet par safe karne ke liye taake PC band hone par bhi data zaya na ho.
